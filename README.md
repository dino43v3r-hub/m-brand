# M Brand

The M Brand repository is the canonical home for the brand system, visual standards, character guidance, assets, templates, and future website for the M Brand.

The repository exists to keep the brand's source-of-truth materials organized, versioned, and easy to extend over time. It supports thoughtful public work across AI, technology, leadership, theology, lifelong learning, and related creative projects.

## Repository Structure

### `docs/`

Contains the canonical brand documentation. This is the primary source of truth for brand identity, principles, voice, content strategy, character standards, visual standards, and archived reference material.

### `assets/`

Contains brand assets, including canonical source assets and derived/exported files. Canonical assets are organized by brand area, such as Dino and ConnectAloud.

### `templates/`

Reserved for reusable brand templates, including future document, presentation, publishing, and content templates.

### `guidelines/`

Reserved for repository-level and operational guidance, such as naming conventions, versioning practices, and contribution workflows.

### `website/`

Reserved for the official Astro website for the M Brand. Astro has not been initialized yet.

## Canonical Documents

Canonical brand documents are located in `docs/`. These files should be treated as the authoritative reference for brand decisions unless formally revised.
